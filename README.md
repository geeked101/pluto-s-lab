# Pluto's Lab 🧪

An interactive React playground for UI experiments, mood-based micro-animations, and a sleek Pomodoro timer.

## ✨ Features
- **Dynamic Mood Switcher**: Toggle between Chill, Dark, and Hype modes with custom gradients and particles.
- **Flying Words**: Animated atmospheric text that reacts to the current mood.
- **Pomodoro Timer**: A minimalist, high-contrast timer for deep focus sessions.
- **Football Cursor**: A custom snowy football cursor that follows the mouse.
- **Gojo Logo**: Featuring Gojo Satoru as the project's official logo.

## 🚀 Deployment

### Vercel (Recommended)
This project is pre-configured for Vercel. 
1. Push to your GitHub repository.
2. Import to Vercel.
3. Vercel will auto-detect the Vite build settings.

### Manual Build
```bash
npm install
npm run build
```
The production-ready files will be in the `/dist` folder.
